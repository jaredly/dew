# dew

# Install

## Requirements

* docker

```
git clone https://github.com/keyvanfatehi/dew `pwd`/.dew-src
sudo ln -sf `pwd`/.dew-src/bin/dew /usr/local/bin/dew
```

# Usage

executing `dew` will print help -- there are only 3 commands at this time: list, install, destroy
